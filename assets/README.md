# Assets 资源目录

这个目录用于存放项目的共享资源文件。

## 目录结构

```
assets/
└── images/          # 图片资源
    ├── screenshots/ # 应用截图（用于文档展示）
    ├── icons/       # 图标资源
    └── banners/     # 横幅图片
```

## 使用说明

### 应用截图
将应用的截图放在 `images/screenshots/` 目录下，用于 README 文档展示。

建议命名规范：
- `flashlight-main.png` - 手电筒应用主界面
- `flashlight-programming.png` - 手电筒编程界面
- `vocabulary-booklist.png` - 单词应用书籍列表
- `vocabulary-study.png` - 单词学习界面

### 图标资源
共享的图标资源可以放在 `images/icons/` 目录下。

### 横幅图片
项目横幅、宣传图等可以放在 `images/banners/` 目录下。

## 注意事项

1. **文件大小**：尽量压缩图片，保持合理的文件大小
2. **命名规范**：使用小写字母和连字符，避免空格和特殊字符
3. **格式选择**：
   - 截图：PNG（保持清晰度）
   - 照片：JPG（较小文件）
   - 图标：PNG（支持透明）
4. **版权**：确保所有资源文件都有使用权限
