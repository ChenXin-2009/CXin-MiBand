## 🎨 压缩策略详解

### 中度压缩（推荐）

```javascript
function compressTranslation(text) {
  return text
    // 1. 删除标记
    .replace(/\[网络\][^\n]*/g, '')
    .replace(/\[地名\][^\n]*/g, '')
    .replace(/\[人名\][^\n]*/g, '')
    
    // 2. 简化词性标记
    .replace(/\bna\.\s*/g, '')
    .replace(/\bn\.\s*/g, '名 ')
    .replace(/\bv\.\s*/g, '动 ')
    .replace(/\badj\.\s*/g, '形 ')
    .replace(/\badv\.\s*/g, '副 ')
    
    // 3. 清理空格
    .replace(/\s+/g, ' ')
    .trim()
    
    // 4. 限制长度
    .substring(0, 150);
}
```

**效果**：
- 删除 `[网络]` 等标记：节省 20-30%
- 简化词性标记：节省 5-10%
- 限制长度：影响 < 0.5% 词条
- **总节省**：约 25-35%

### 示例

n. 苹果
