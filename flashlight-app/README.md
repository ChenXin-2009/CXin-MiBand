# 手电筒应用 - 可编程版本

一个功能强大的手电筒应用，支持自定义颜色和可视化编程，适用于小米手环/手表 Vela OS。

## ✨ 主要特性

### 🎨 颜色控制
- **RGB精确调整**：支持0-255全范围调整
- **常用颜色**：9种预设颜色快速选择（红、绿、蓝、青、紫、黄、白、橙、粉）
- **实时预览**：调整时即时查看效果
- **最高亮度**：自动设置最高亮度以获得最佳效果

### 💡 屏幕常亮
- 手电筒开启时自动保持屏幕常亮
- 退出时自动恢复原始设置
- 确保手电筒持续工作

### 🎯 可视化编程
- **拖拽式编辑**：直观的指令管理
- **四种指令**：切换颜色、熄灭、亮起、等待
- **实时测试**：保存前可测试程序效果
- **本地存储**：程序保存在手表本地

### 🚨 默认程序
- **白光SOS**：标准国际求救信号（... --- ...）
- **红蓝交替**：警车灯效果
- **快速闪烁**：50ms高频频闪

## 📱 应用界面

### 主界面
- 颜色预览区域
- 常用颜色快速选择
- RGB精确调整（+10/-10, +1/-1）
- 编程功能入口

### 编程界面
- 程序列表（默认 + 自定义）
- 程序编辑器（可视化指令管理）
- 指令管理（上移、下移、删除）
- 程序执行（循环运行）

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0
- 小米手环/手表（支持 Vela OS）
- 最低平台版本：1070

### 安装依赖
```bash
cd vela-flashlight-app
npm install
```

### 开发调试
```bash
# 开发模式（带热更新）
npm run dev

# 或使用 AIoT-toolkit
npx aiot-toolkit start
```

### 构建发布
```bash
# 构建项目
npm run build

# 打包发布
npm run release
```

打包后的 .rpk 文件位于：`dist/com.flashlight.app.rpk`

### 安装到手环
1. 将打包好的.rpk文件传输到手机
2. 打开小米运动健康App
3. 进入"设备" → 选择手环 → "应用管理"
4. 点击"添加应用"，选择.rpk文件
5. 等待安装完成

## 📖 使用指南

### 基础使用

#### 1. 使用固定颜色
1. 在主页面选择或调整颜色
2. 点击"确认颜色"按钮
3. 手电筒以选定颜色亮起
4. 按返回键退出

#### 2. 使用默认程序
1. 点击"📝 编程"按钮
2. 在默认程序中选择一个（SOS/红蓝交替/快速闪烁）
3. 程序开始循环执行
4. 按返回键退出

### 高级使用

#### 创建自定义程序
1. 点击"📝 编程" → "➕ 新建程序"
2. 输入程序名称和描述
3. 添加指令：
   - **🎨 切换颜色**：选择RGB值（支持常用颜色和精确调整）
   - **⏱️ 等待**：设置延迟时间（支持+100/-100ms, +10/-10ms, +1/-1ms）
   - **⚫ 熄灭**：关闭手电筒
   - **⚪ 亮起**：恢复上次颜色
4. 调整指令顺序（使用↑↓按钮）
5. 点击"▶️ 测试运行"预览效果
6. 点击"💾 保存程序"

#### 编辑程序
1. 在程序列表中点击自定义程序
2. 修改名称、描述或指令
3. 保存或删除程序

### 数据存储格式
```javascript
{
  key: 'customPrograms',
  value: [
    {
      id: '1234567890',           // 时间戳作为唯一ID
      name: '程序名称',
      description: '程序描述',
      commands: [
        {
          type: 'color',          // 颜色指令
          r: 255, g: 0, b: 0,
          display: '🎨 颜色: RGB(255, 0, 0)'
        },
        {
          type: 'wait',           // 等待指令
          time: 1000,
          display: '⏱️ 等待: 1000ms'
        },
        {
          type: 'off',            // 熄灭指令
          display: '⚫ 熄灭'
        },
        {
          type: 'on',             // 亮起指令
          display: '⚪ 亮起 (恢复上次颜色)'
        }
      ]
    }
  ]
}
```

## ⚠️ 注意事项

1. **电量消耗**：长时间使用会快速消耗电量，特别是高亮度和快速闪烁模式
2. **时间精度**：等待时间受系统定时器限制，可能有±10ms误差
3. **存储限制**：建议自定义程序数量不超过20个
4. **指令数量**：建议单个程序指令数量不超过50条
5. **极短延迟**：小于10ms的等待时间可能无法准确执行
6. **屏幕保护**：程序执行时会保持屏幕常亮，长时间使用请注意
