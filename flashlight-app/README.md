# 手电筒应用 - 可编程版本

一个功能强大的手电筒应用，支持自定义颜色和可视化编程，适用于小米手环/手表 Vela OS。

## ✨ 主要特性

### 🎨 颜色控制
- **即开即亮**：进入应用立即全屏亮起上次使用的颜色
- **点击进入设置**：点击屏幕进入颜色选择界面
- **RGB精确调整**：支持0-255全范围调整（+1/-1, +10/-10）
- **常用颜色**：9种预设颜色快速选择（红、绿、蓝、青、紫、黄、白、橙、粉）
- **实时预览**：调整时即时查看效果
- **最高亮度**：自动设置最高亮度以获得最佳效果
- **持久化存储**：颜色设置自动保存，下次打开直接使用

### 💡 屏幕常亮
- 手电筒开启时自动保持屏幕常亮
- 退出时自动恢复原始设置
- 确保手电筒持续工作

### 🌈 跑马灯
- **色相环渐变**：HSV色相环循环变化，产生彩虹流光效果
- **左滑进入设置**：跑马灯页面左滑进入设置页
- **速度可调**：慢(100ms) / 中(50ms) / 快(20ms)
- **步进可调**：细(1度) / 中(3度) / 粗(6度)
- **设置持久化**：速度与步进设置自动保存


### 🎯 可视化编程
- **拖拽式编辑**：直观的指令管理
- **四种指令**：切换颜色、熄灭、亮起、等待
- **实时测试**：保存前可测试程序效果
- **本地存储**：程序保存在手表本地

### 🚨 默认程序
- **白光SOS**：标准国际求救信号（... --- ...）
- **红蓝交替**：警车灯效果
- **快速闪烁**：50ms高频频闪



## 📊 交互流程

```
打开应用
  → 全屏灯亮起（上次颜色）
    ├── 点击屏幕 → 颜色选择页
    │   ├── 调整颜色
    │   └── 确认 → 返回全屏灯
    └── 左滑 → 功能选择页

            ├── 默认程序（SOS/红蓝交替/快速闪烁）
            ├── 新建程序 → 程序编辑器
            └── 编辑程序 → 程序编辑器
```


## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0
- 小米手环/手表（支持 Vela OS）
- 最低平台版本：1070

### 安装依赖
```bash
cd vela-flashlight-app
npm install
```

### 开发调试
```bash
# 开发模式（带热更新）
npm run dev

# 或使用 AIoT-toolkit
npx aiot-toolkit start
```

### 构建发布
```bash
# 构建项目
npm run build

# 打包发布
npm run release
```

打包后的 .rpk 文件位于：`dist/com.flashlight.app.rpk`

### 安装到手环
1. 将打包好的.rpk文件传输到手机
2. 打开小米运动健康App
3. 进入"设备" → 选择手环 → "应用管理"
4. 点击"添加应用"，选择.rpk文件
5. 等待安装完成

## 📖 使用指南

### 基础使用

### 1.固定颜色
1. 打开应用，手电筒自动亮起（上次使用的颜色）
2. 点击屏幕进入颜色选择页
3. 选择预设颜色或微调RGB值
4. 点击"确认颜色"，返回全屏灯

### 2.跑马灯
1. 在全屏灯页面**左滑**，进入功能选择页
2. 点击"跑马灯"
3. 色相环渐变色自动循环播放
4. 左滑进入设置页调整速度与步进
5. 右滑逐级返回

### 高级使用

#### 创建自定义程序
1. 点击"📝 编程" → "➕ 新建程序"
2. 输入程序名称和描述
3. 添加指令：
   - **🎨 切换颜色**：选择RGB值（支持常用颜色和精确调整）
   - **⏱️ 等待**：设置延迟时间（支持+100/-100ms, +10/-10ms, +1/-1ms）
   - **⚫ 熄灭**：关闭手电筒
   - **⚪ 亮起**：恢复上次颜色
4. 调整指令顺序（使用↑↓按钮）
5. 点击"▶️ 测试运行"预览效果
6. 点击"💾 保存程序"

#### 编辑程序
1. 在程序列表中点击自定义程序
2. 修改名称、描述或指令
3. 保存或删除程序

### 数据存储格式
```javascript
{
  key: 'customPrograms',
  value: [
    {
      id: '1234567890',           // 时间戳作为唯一ID

      description: '程序描述',

        {

          display: '🎨 颜色: RGB(255, 0, 0)'
        },
        {
          type: 'wait',           // 等待指令
          time: 1000,

        },
        {
```

## ⚠️ 注意事项

1. **电量消耗**：长时间使用会快速消耗电量，特别是高亮度和快速闪烁模式
2. **时间精度**：等待时间受系统定时器限制，可能有±10ms误差
3. **存储限制**：建议自定义程序数量不超过20个
4. **指令数量**：建议单个程序指令数量不超过50条
5. **极短延迟**：小于10ms的等待时间可能无法准确执行
6. **屏幕保护**：程序执行时会保持屏幕常亮，长时间使用请注意
