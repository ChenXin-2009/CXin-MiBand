# 单词背诵应用 (Vocabulary App)

一个为小米手环/手表开发的高中英语单词学习应用，支持按教材单元学习，包含完整的必修和选择性必修教材词汇。

## ✨ 主要特性

### 📚 完整教材覆盖
- **必修系列**：必修第一册（含Welcome Unit）、第二册、第三册
- **选择性必修系列**：选择性必修第一册、第二册、第三册、第四册
- **总词汇量**：36个单元，共2645个单词

### 📖 灵活学习模式
- **按顺序学习**：按照教材顺序系统学习
- **随机学习**：打乱顺序，增强记忆效果
- **单元选择**：可以选择任意单元进行学习

### 💡 简洁交互设计
- **第一次点击**：显示音标和中文释义
- **第二次点击**：进入下一个单词
- **右滑返回**：任何页面都支持右滑返回
- **自动完成**：学完所有单词后自动返回

### 💾 本地存储
- 所有单词数据存储在本地
- 无需网络连接
- 快速响应，流畅体验

## 📱 应用界面

### 1. 书籍选择页面
显示所有可用的教材书籍，包括必修和选择性必修系列。

### 2. 单元选择页面
显示选定书籍的所有单元（Unit 1-5，必修第一册还包含Welcome Unit）。

### 3. 模式选择页面
选择学习模式：
- 📖 按顺序：系统学习
- 🔀 随机：强化记忆

### 4. 学习页面
- 显示英文单词
- 点击显示音标和释义
- 再次点击进入下一个单词
- 显示学习进度（如：1/50）

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0
- 小米手环/手表（支持 Vela OS）
- 最低平台版本：1070

### 安装依赖
```bash
cd vocabulary-app
npm install
```

### 开发调试
```bash
# 开发模式（带热更新）
npm run dev

# 或使用 AIoT-toolkit
npx aiot-toolkit start
```

### 构建发布
```bash
# 构建项目
npm run build

# 打包发布
npm run release
```

打包后的 .rpk 文件位于：`dist/com.vocabulary.app.rpk`

### 安装到手环
1. 将打包好的.rpk文件传输到手机
2. 打开小米运动健康App
3. 进入"设备" → 选择手环 → "应用管理"
4. 点击"添加应用"，选择.rpk文件
5. 等待安装完成

## 📖 使用指南

### 基本使用流程

1. **选择书籍**
   - 打开应用，进入书籍选择页面
   - 滑动浏览所有可用教材
   - 点击选择要学习的书籍

2. **选择单元**
   - 浏览该书籍的所有单元
   - 点击选择要学习的单元

3. **选择模式**
   - **按顺序**：适合第一次学习，系统掌握
   - **随机**：适合复习巩固，打乱顺序加深记忆

4. **开始学习**
   - 屏幕显示英文单词
   - 点击一次：显示音标和中文释义
   - 再点击一次：进入下一个单词
   - 右滑：返回上一级
   - 学完所有单词后自动返回模式选择页面

### 学习技巧

1. **首次学习**：使用"按顺序"模式，系统学习
2. **复习巩固**：使用"随机"模式，检验记忆效果
3. **分段学习**：每次学习一个单元，避免疲劳
4. **重复学习**：对不熟悉的单元多次学习

```

├── src/
│   ├── pages/
│   │   ├── book-list/          # 书籍选择页面
```

## 🛠️ 开发指南

### 更新单词数据

如果需要更新或添加单词数据：

1. 编辑 `data/csv_by_unit/` 目录下的CSV文件
2. CSV格式：`单词,音标,释义`
3. 运行转换脚本：
   ```bash
   node convert-csv-to-json.js
   ```
4. 重新构建应用

### CSV文件命名规范
- 格式：`{书籍名}_{单元名}.csv`
- 示例：`必修第一册_unit_1.csv`
- 特殊：`必修第一册_welcome.csv`

## ⚠️ 注意事项

1. **存储空间**：手环存储空间有限，2645个单词已接近上限
2. **性能优化**：大量数据加载时可能有短暂延迟
3. **字体显示**：部分特殊音标符号可能显示效果因设备而异
4. **数据备份**：更新数据前建议备份原CSV文件
