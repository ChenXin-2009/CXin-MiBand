# 单词背诵应用

这是一个为小米手环开发的英语单词背诵应用，支持多本教材的单词学习。

## 功能特点

1. **多书籍支持**：包含必修第一册到第三册，以及选择性必修第一册到第四册
2. **单元选择**：每本书可以选择不同的Unit进行学习
3. **学习模式**：
   - 按顺序：按照单词表的顺序学习
   - 随机：随机打乱单词顺序学习
4. **交互方式**：
   - 第一次点击：显示单词的音标和中文释义
   - 第二次点击：进入下一个单词
5. **右滑返回**：在任何页面都可以右滑返回上一级

## 使用流程

1. 打开应用，进入书籍选择页面
2. 选择要学习的书籍（如"必修第一册"）
3. 选择要学习的Unit（如"Unit 1"）
4. 选择学习模式（按顺序或随机）
5. 开始背单词：
   - 屏幕显示英文单词
   - 点击一次，显示音标和中文释义
   - 再点击一次，显示下一个单词
   - 学完所有单词后自动返回

## 数据来源

单词数据来自 `csv_by_unit` 文件夹中的CSV文件，通过 `convert-csv-to-json.js` 脚本转换为JSON格式。

## 开发说明

### 转换CSV数据

如果CSV文件有更新，运行以下命令重新转换：

```bash
node convert-csv-to-json.js
```

### 编译和安装

```bash
npm run build
```

然后通过小米手环开发工具安装到设备上。

## 文件结构

```
src/
├── pages/
│   ├── book-list/      # 书籍选择页面
│   ├── unit-list/      # Unit选择页面
│   ├── mode-select/    # 模式选择页面
│   └── study/          # 背单词页面
├── data/
│   └── words.json      # 单词数据（由CSV转换而来）
└── manifest.json       # 应用配置
```
